系统简介

社脉通 是一个基于关系数据库的社会关系网络穿透分析系统，采用 Python Flask + SQLite + ECharts 技术栈构建。

主要功能：人员与组织信息的多维存储、社会关系与职责管理、全局关系网络可视化、多层关系穿透分析，以及三级用户权限控制。

技术特色：遵循3NF规范化设计、外键级联约束、BFS穿透分析算法、ECharts力导向图可视化。
