# 社脉通 - 社会关系网络穿透分析系统 需求分析文档

## 1. 引言

### 1.1 编写目的
本文档旨在明确"社脉通"社会关系网络穿透分析系统的功能需求和非功能需求，为后续的设计、开发和测试提供依据。

### 1.2 项目背景
在社会治安管理、社区网格化治理、企业人力资源分析等场景中，人员之间的关系网络是一项重要的分析对象。传统的关系管理依赖纸质档案或简单的电子表格，无法直观呈现关系网络，更无法进行深层关系挖掘。本系统旨在解决这一问题，提供一个集数据存储、可视化展示、关系穿透分析于一体的Web应用。

### 1.3 适用范围
- 社区网格员/民警：用于辖区人员关系管理
- 企业HR：用于员工关系网络分析
- 校友会：用于校友关系追踪
- 课程教学：作为数据库课程设计的完整实践案例

## 2. 系统总体描述

### 2.1 系统目标
构建一个基于关系数据库的社会关系网络分析系统，支持人员信息、组织信息、社会关系和社会职责的多维存储，实现关系网络的可视化展示和深层关系穿透分析。

### 2.2 用户角色

| 角色 | 级别 | 权限说明 |
|------|------|----------|
| 普通大众 | 1 | 可查看人员和组织的基本公开信息，浏览关系网络图 |
| 民警 | 2 | 可查看详细信息，进行关系穿透分析查询 |
| 高级管理员 | 3 | 拥有全部增删改查权限，可管理用户权限 |

### 2.3 运行环境
- 操作系统：Windows / Linux / macOS
- 运行环境：Python 3.8+
- 数据库：SQLite（内置，无需额外安装）
- 浏览器：Chrome / Edge / Firefox 等现代浏览器
